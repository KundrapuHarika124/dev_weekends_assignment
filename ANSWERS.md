# Assessment Answers

## 1. How to run

```bash
npm install
npm run dev
```

## 2. Stack & design choices

- React for component-based UI.
- Vite for fast local development and production builds.
- Tailwind CSS for concise responsive styling.

## 3. Responsive & accessibility decisions

- The layout stacks on smaller screens and switches to two columns on desktop.
- Inputs are labeled and error messages are linked with ARIA attributes.
- Buttons and inputs include visible focus states for keyboard navigation.

## 4. AI usage honestly

This project was assembled with GitHub Copilot assistance.

## 5. One thing not fully polished

There is no persistent storage yet, so the form resets on refresh.
